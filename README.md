# Code Mentor 
### Alert ->(the name is changed from Student-Support to Code Mentor)

## Description
Code Mentor is the Platform to Support Students and buniness to buy the readymade Projects and they can book the session to debug the
existing Project to make it works for them and also they can post the ideas we varify them and Build for them for there requirement,
and one more is Project navigator that is to help the student to guild from starting to end of there Project to tell them which tech stack
is good fit for you Project and where to host it this all guidance they can get from expert.

## Features
- Authantication & Authrization 
- Context API fro Global state management
- Admin panal
- Role Base Access
- CRUD Operations

## Tech Stack 
- React js 
- Tailwindcss
- Express js
- Node js 
- MongoDB
- Framer Motion

## Installation 
```bash 
git clone https://github.com/SudeepNPatil/Student-Support.git
cd Student-Support
cd Frontend
npm install
npm run dev
for backend in new terminal
cd backend
npm install 
npm start

Create a .env file in the Backend and add:
DB_URL = your-Db_url_here
JWT= your_JWT_Screate_here
PORT = Port_you_want_to_Run
```
